Bike Sharing Demand

Machine Learning

This is the final project for my data science class and part of a Kaggle competion. The purpose of the project was to forecast bike rental demand for the city of Washington D.C.. For this we ran several machine learning(ML) algorithms and techniques on a training dataset and built several ML models. We then used our models and ran it against a testing dataset to attain a prediction accuracy. The highest accuracy achieved was 79.7%. Given the complexity of the dataset we achieved great results. The project was implemented using Python 3, scikit-learn, numPy, pandas, and Jupyter Notebook.

MINE

Web Service

Prototype for Medical Indexing Enhancement web service(MINE). The web service allows a user to choose a file and load it on a custom PDF viewer. The user then tags each page with one or more tag that we provided. When the user is done going throught the whole document they hit the group button, this creates a new sorted PDF document sorted according to the priority of each label.A table of index is also created with the new sorted pages noted. The prototype is currently being reviewed by management, if approved database implementation will follow. The web service was built with ASP.NET and PDF SDK Syncfusion. Unfortanely due to security reasons code cannnot be shared or viewed.
